Proefnotitie

100 schone PDF's verwerkt met zichtbaar runbewijs.

Deze interne run ondersteunt Solvyrs schone digitale PDF/tekstpad: extractie, chunking, embeddings en een zichtbare runsamenvatting. Het is geen claim voor OCR-zware scans, layoutparsing of gestructureerde extractie.

Gemeten run

Wat de run liet zien.

De waarde van deze proef zit niet in een grote benchmarkclaim. Het is een concrete outputgrens: input, output, looptijd, fouten, retries en caveats zijn inspecteerbaar.

Input 100 PDF's / 2.052 pagina's
Uitvoering 215s looptijd / 0,281 GPU-uren gemeten
Output 987 chunks / ongeveer 1,3M geschatte tokens
Betrouwbaarheid 0 fouten / 0 retries

Wat dit ondersteunt

  • Corpusvoorbereiding voor schone digitale PDF/tekst.
  • Chunking en embeddings als onderdeel van een batchpad.
  • Runsamenvattingen met zichtbare output en caveats.
  • Een gecontroleerd pilotgesprek rond vergelijkbare workloads.

Wat dit niet claimt

  • Geen OCR-zware scanondersteuning standaard.
  • Geen claim voor gestructureerde factuur- of formulierextractie.
  • Geen hosted RAG/search productclaim.
  • Geen enterprise-SLA claim.
