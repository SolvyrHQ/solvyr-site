Use case

Turn a PDF backlog into an AI-ready RAG corpus.

Solvyr Fast Corpus converts clean digital PDF backlogs into JSONL with extracted text, chunks, embeddings, stable IDs, and a run report for RAG, search, evals, and agent workflows at a predictable starting price.

Scope a PDF backlog run See sample output

Use this when

  • You have a backlog of clean digital PDFs.
  • You need chunks, embeddings, and stable source IDs.
  • You want an artifact your AI team can inspect before indexing.
  • You want a cost-effective run before building another pipeline.
  • The workload can run asynchronously.

Not for

  • OCR-heavy scans unless scoped separately.
  • Realtime chat serving or model hosting.
  • Custom structured extraction in the current trial.
  • Highly sensitive data without prior agreement.

Output

What a PDF-to-RAG run returns.

The expected bundle includes extracted text, chunks, embeddings, stable IDs, item-level outcome notes where applicable, and a run report with counts, caveats, and price basis. Standard clean PDF/text processing starts at EUR 0.95 per 1000 pages, excl. VAT.
